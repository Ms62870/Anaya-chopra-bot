# Anaya AI Companion Bot

Anaya AI is a Telegram AI Companion Bot powered by Google Gemini.

## Features

- Human-like AI conversation
- User profiles
- Conversation memory
- Premium subscription
- Admin panel
- Railway deployment

## Tech Stack

- Python 3.12
- Aiogram 3
- Google Gemini API
- Railway
- SQLite (PostgreSQL later)
