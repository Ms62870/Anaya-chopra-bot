# 🤖 Anaya AI Companion Bot

Anaya AI is a Telegram AI Companion Bot powered by Google Gemini.

## ✨ Features

- Human-like AI conversation
- Conversation memory
- User profile
- Premium membership
- Admin panel
- Telegram bot support
- Railway deployment

## 🛠️ Tech Stack

- Python 3.12
- Aiogram 3
- Google Gemini API
- Railway
- PostgreSQL (later)
- SQLite (development)

## 📁 Project Structure

```
app/
    main.py

    README.md
    requirements.txt
    Procfile
    runtime.txt
    railway.json
    .env.example
    .gitignore
    ```

    ## 🚀 Deployment

    1. Create Telegram Bot using BotFather.
    2. Add `BOT_TOKEN` in Railway Variables.
    3. Add `GEMINI_API_KEY` in Railway Variables.
    4. Deploy on Railway.
